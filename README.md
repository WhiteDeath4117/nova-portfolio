# NOVA – Personal Portfolio Website

A futuristic, interactive portfolio website built to showcase my skills, projects, and growth as an aspiring developer and digital creator.

## ✨ Features

- **Immersive 3D Background** – Particle field and animated torus knot using Three.js
- **Custom Cursor** – Glowing dot + outline with magnetic hover effects
- **Smooth Scroll** – Buttery-smooth navigation with Lenis
- **Typed.js Effect** – Dynamic text rotation in the hero section
- **Animated Skills Section** – Progress bars with scroll-triggered GSAP animations
- **Expandable Journey Timeline** – Click to reveal details of your learning path
- **Project Cards** – Tilt effect with glassmorphism design
- **3D Avatar** – Animated floating head with glowing eyes and visor
- **Responsive Design** – Works on desktop and mobile
- **Loading Screen** – Gradient‑text “NOVA” with progress bar
- **Contact Form** – Ready for inquiries (add backend later)

## 🛠️ Tech Stack

- **HTML5** / **CSS3** (custom properties, grid, glassmorphism)
- **JavaScript** (ES6+)
- **Three.js** – 3D graphics
- **GSAP** + **ScrollTrigger** – smooth animations
- **Lenis** – smooth scrolling
- **Typed.js** – typing animation
- **Vanilla Tilt** – 3D tilt effects
- **Font Awesome** – icons
- **Google Fonts** – Plus Jakarta Sans, Space Grotesk, Inter

## 🖥️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/WhiteDeath4117/nova-portfolio.git
   ```
2. Open `NOVA.html` (or rename to `index.html`) in any modern browser.
3. No build steps or dependencies needed – everything loads via CDN.
